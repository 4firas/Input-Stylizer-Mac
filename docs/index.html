so the app doesnt come with a brain. u gotta point it at one. it works with anything that does the openai chat thing which is like. most of them

built in: lm studio (free, runs on ur computer), openrouter (cheap, lots of models), openai (u know it), opencode go (also cheap, lots of models), or custom url if u got one

if u wanna go full tin foil hat, point it at http://localhost:1234/v1 and ur good. no cloud. no logging. no anything

how 2 install (its fast)

  1. go to the releases page and grab SystemWideStylizer.app.zip
  2. unzip it. drag the .app to applications like any other app
  3. open it. mac will be like "this is from an unidentified developer" and u say yes. system settings > privacy > security > open anyway
  4. go to system settings > privacy & security > accessibility and turn the app on. this is the weird part where mac gets weird about it
  5. click the little dotted-line abc thing in ur menu bar. pick a style and a provider. paste ur api key if u need one
  6. go type anywhere. hit enter. watch the magic happen (or the chaos depending on the style lol)

[tip: if the menu bar icon looks grayed out, accessibility isnt on yet. go turn it on in system settings. u only gotta do it once per mac, then it just works forever]

FAQ (questions ppl actually asked me)

does it like, send everything i type to someone??
only to the ai u picked in settings. thats it. nothing else. no analytics no tracking no "phone home" stuff. the app is open source so u can literally read every line and confirm. i pinky promise
does it work in [some app]
if it has a text box u can type in then yeah probably. works in discord, slack, imessage, notes, browsers, terminal, basically anything that isnt a password field. password fields it leaves alone bc im not a monster
why is the menu bar icon gray
u dont have accessibility turned on for it yet. system settings > privacy > accessibility > toggle it on. the icon lights up blue when its good to go
can i make my own style??
yes!! pick "custom prompt" from the style menu and write whatever u want. want ur messages to read like a guy named jeff? an instagram caption? a wikipedia article? an apology letter? an email from a corporate ceo? go nuts i wont judge
will apple kill this in a future mac update
idk maybe? the accessibility stuff has been stable since like 2014 tho. if they break it ill find another way. probably. dont quote me on that lol
why the dotted underline icon
bc its an SF symbol called textformat.abc.dottedunderline and it is literally the perfect icon for this. idc if u think its weird. its staying
why did u even make this
bc i was tired of typing "ok thanks" and wishing it said more. also i thought it would be funny. it is. try the pirate one on ur boss and report back pls i need to know if it gets u fired
