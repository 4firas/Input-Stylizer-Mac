how_to_install.txt

install (its fast i promise)

  1. go to the releases page and grab SystemWideStylizer.app.zip
  2. unzip it, drag the .app into ur applications folder like any other app
  3. open it. mac will be like "this is from an unidentified developer, are u sure" and u say yes. system settings > privacy > security > open anyway
  4. go to system settings > privacy & security > accessibility and turn the app on. this is the part where mac gets weird about it
  5. click the little dotted-line abc thing in ur menu bar. pick a style and a provider. paste ur api key if u need one
  6. go type anywhere. hit enter. watch the magic happen (or the chaos, depending on the style)

if the menu bar icon looks grayed out, the app doesnt have accessibility yet. go turn it on in system settings. u only gotta do this once per mac
